## TESTING PLAN

**Phase 1 (Week 1): Test angles**
- Run all 6 angles with $10/day each = $60/day total
- Same landing page for all
- Measure: CTR, quiz starts, quiz completions, leads

**Phase 2 (Week 2): Double down on winners**
- Kill bottom 3 angles
- Scale top 3 to $20/day each
- Start testing ad creatives (image vs video)

**Phase 3 (Week 3+): Optimise**
- Auto-anneal funnel based on drop-off data
- Test headline variants on landing page
- Scale winning angle to $50-100/day
